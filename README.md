# Ultimate Scientific Calculator

A fully functional **Ultimate Scientific Calculator** built with **HTML, CSS, and JavaScript**.  
This calculator supports both basic and advanced mathematical operations, including trigonometry in degrees, factorials, powers, fractions, memory functions, and more. It also includes a **dark mode toggle** and a **scientific button toggle** for a clean interface.

---

## Features

### Basic Functions
- Addition (+), Subtraction (-), Multiplication (*), Division (/)
- Decimal support
- Parentheses for order of operations
- Clear (C) and Delete (DEL) buttons

### Scientific Functions
- Trigonometric functions: `sin`, `cos`, `tan`, `cot`, `sec`, `csc` (in degrees)
- Inverse trig: `asin`, `acos`, `atan`
- Hyperbolic functions: `sinh`, `cosh`, `tanh`, `coth`, `sech`, `csch`
- Powers and roots: `x^y`, `‚àö`, `‚àõ`
- Factorials, combinations (`nCr`), permutations (`nPr`)
- Logarithms: `ln`, `log10`
- Exponential: `exp`, constants: `œÄ`, `e`
- Fractions to decimals and decimals to fractions

### Additional Features
- **Memory functions:** M+, M-, MR, MC
- Dark mode toggle
- Toggle scientific buttons
- Keyboard support (numbers, operators, Enter, Backspace, Escape)
- Responsive and stylish UI using CSS and Font Awesome
